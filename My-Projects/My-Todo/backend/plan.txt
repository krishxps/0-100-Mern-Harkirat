Register : class - username , password , isAdmin
Login : class - username , password
Todo: class - title , description , completed , created , deadline

User : 
- Can Register
- Can Login
- Can Create Todo
- Can Update Todo
- Can Delete Todo
- Can View Todo

Admin : 
- Can Register
- Can Login
- Can Create Todo
- Can Update Todo
- Can Delete Todo
- Can View Todo
- Can Delete User
